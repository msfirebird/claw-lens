# Changelog

## v0.1.0

Initial release.

- Local dashboard at `http://localhost:4242`
- Cost monitoring
- Session browser with cost, duration, model, and error breakdown
- Tool profiler with call count, failure rate, and p95 latency
- Token usage tracking with cache efficiency metrics
- Live monitor via WebSocket proxy to OpenClaw Gateway
- Security audit with sensitive data scanning and risk scoring
- Memory viewer for agent workspace files
- Cron session detection and grouping
- CLI support: `--port`, `--no-open`
